# Lean4Web all-block paste files

The directory `all560_blocks/` contains self-contained Lean files for all
`560` blocks:

```text
all560_blocks/Block000_lean4web.lean
...
all560_blocks/Block559_lean4web.lean
```

Each file can be pasted into Lean4Web independently.  Passing one file means
that Lean accepted the rational-manifest check for that individual block.

These files were generated by:

```bash
python3 scripts/make_all560_lean4web_exports.py
```

The import-based all-block aggregate theorem is checked locally with:

```bash
scripts/check_pure_lean.sh
```

Each block file ends with a `#check` line of the following form:

```lean
Erdos1038Lean.M1817475.Block000.block000_rational_certificate_proof :
  Erdos1038Lean.M1817475.Block000.block000_rational_certificate
```

## Caveat

These files are still the rational-manifest layer.  They do not yet contain the
Mathlib `Real.log` bridge proving that the submitted lower-bound rationals are
actually lower bounds for the logarithmic potential.
